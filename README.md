# Brand Assets

Official brand assets for Bitcast and Stitch3.

## Structure

```
brand/
├── logo/        # Full logos (icon + wordmark)
├── icon/        # Standalone icons/marks
├── wordmark/    # Text-only logos
└── colors/      # Color palettes and swatches
```

Each asset directory contains source SVGs and any exported variants.

## Brands

- **[Bitcast](bitcast/)** — Decentralized marketing infrastructure on Bittensor
- **[Stitch3](stitch3/)** — Creator campaign platform by Bitcast
